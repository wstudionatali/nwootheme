# nwootheme start theme for woocommerce

## Features

- Custom Home page.
- Single Product Page
- Custom Cart Page
- Custom Checkout Page

## Uses

- TailwindCSS
- Webpack
- Babel
- Sass

## Maintainer

1. clone in wordpress theme folder
2. Make sure WooCommerce plugin is active.


## Development ( To be added )

**Install**

Clone the repo and run

```bash
cd nwootheme/assets
npm update 
```

**During development**

```bash
cd assets

```

**Production**

```bash
cd assets
```
